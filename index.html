Bootstrap 4

A quick summary

For the foreseeable future, they’ll be maintaining Bootstrap 3 with critical bug fixes and documentation improvements.

Bootstrap throughout the years

  • 1.0 grids, typography and lists, not much else
  • 2.0 a fully responsive framework!!
  • 3.0 flat design, new features and bug fixes
  • 4.0 more features, less old browser supporting!

Revamped grid

IE8 support dropped

  • Media-queries supported
  • Better CSS3 support
  • JQuery 2.x as default
  • Flexbox support

Also: iOS 6 support dropped, Android v5.0 Lollipop’s Browser and WebView now officially supported.

Flexbox!

CSS

  • Generated from SCSS instead of LESS
  • Rewritten to use less specific CSS selectors
  • Typography now rem and em-based.
  • Tons of utility classes around padding and margins
  • Neat toggles

333 SCSS variables

Components

  • No more glyphicons
  • New .btn-outline style
  • Nicer styling via WTFForms library for checkboxes, radio buttons, select menus, file inputs, and progress bars
  • wells panels thumbnails cards!

Cards

JavaScript

  • Supports jQuery 2
  • UMD (Universal Module Definition) pattern support
  • All plugins re-written using ES6
  • Dropped Affix plugin
  • Better positioning for tooltips, popovers, and dropdowns.

Next

  • A few alpha releases while things are still in flux.
  • Two beta releases after features and functionality are locked down to really test things out.
  • Two release candidates (RCs) to really test things out closer to production environments.
  • Then, the final release!

Get it

  • Documentation: http://v4-alpha.getbootstrap.com/

    • CSS at https://cdn.rawgit.com/twbs/bootstrap/v4-dev/dist/css/bootstrap.css
